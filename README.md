# vue-resize

## 使用方法

### <Resize>xxx</Resize>

## Attributes
### | 参数 | 说明 | 类型 | 默认值 |
### | min | 最小宽高 | object | {width: 0, height: 0} |
### | max | 最大宽高 | object | {width: 0, height: 0} |
### | move | 移动手柄配置 | object | {t: true, l: true, r: true, b: true, tl: true, tr: true, bl: true, br: true} |
### | disabled | 是否禁用 | object | false |
